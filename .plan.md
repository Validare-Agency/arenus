# Arenus Desktop Header Implementation Plan

## Summary
Replace the existing header with a new Arenus-branded two-row header based on the Figma design: a thin top utility bar + a thicker main header bar, both on a dark (#191919) background.

## Figma Design Recap

**Row 1 (Top Bar):** Dark bg, border-bottom (#45484a). Right-aligned: "Arenus Ambassadors" link, "Become a Dealer" link, country selector (US flag + "US" + caret dropdown with US/Canada options). 12px text, #ddd color.

**Row 2 (Main Header):** Dark bg. LEFT: Logo + two pill buttons ("Colic Assurance Program" with medal icon, "Our Blog" with newspaper icon) — translucent pills with backdrop-blur. RIGHT: Navigation menu links + search/user/cart icons. Cart badge is gold (#e6b236).

---

## Implementation Steps

### Step 1: Create new SVG icon assets
**Files:** `assets/icon-medal.svg`, `assets/icon-newspaper.svg`, `assets/icon-caret-down.svg`
- Extract/create simple SVG icons for the medal and newspaper pill buttons, and the caret-down for the country selector
- These are needed for the pill buttons and country dropdown in the new design

### Step 2: Create the top utility bar snippet
**File:** `snippets/header-top-bar.liquid`
- New snippet for the thin utility bar above the main header
- Contains: configurable link items (Ambassadors, Dealer) + country selector dropdown (US/Canada)
- Country selector is a simple custom JS dropdown (NOT Shopify's localization), toggled on click
- All text, links, and visibility are driven by section settings passed from header.liquid
- Inline `{% stylesheet %}` for top bar styling

### Step 3: Create the header pill buttons snippet
**File:** `snippets/header-pills.liquid`
- Renders the two translucent pill buttons (Colic Assurance Program, Our Blog)
- Each pill: icon (via inline SVG) + text label, wrapped in an `<a>` tag
- Text, link, and icon are driven by section block settings
- Styled with backdrop-blur, translucent bg (rgba(51,61,66,0.4)), 6px radius

### Step 4: Rewrite `sections/header.liquid`
**Major changes:**
- **HTML Structure:** Replace the two-row grid system with a simpler structure:
  1. Render `header-top-bar` snippet as the first row
  2. Main header row with: left side (logo block + pills), right side (menu + icons)
- **Keep existing:** `<header-component>`, sticky/transparent logic, `content_for 'block'` for logo and menu, `render 'header-actions'`, `render 'search'`
- **Remove:** The generic top/bottom row loop, localization markup (replaced by custom country selector in top bar), the `header-row` snippet usage for the main header (we'll inline the new layout)
- **CSS (`{% stylesheet %}`):** Rewrite to match the dark Arenus design:
  - Dark background (#191919) for both rows
  - White text/icons
  - Gold cart badge (#e6b236 from `settings.brand_congress_blue_default`)
  - Proper spacing (40px horizontal padding, 20px vertical for main row, 8px for top bar)
  - Pill button styles (translucent bg, backdrop-blur)
  - Navigation link styles (14px, white, 20px gap)
  - Icon sizing and spacing (24px icons, 8px gap)
- **Schema:** Add new settings for:
  - Top bar: link text/URL for each item, country selector toggle
  - Pill buttons: text/URL/icon for each pill (as blocks or section settings)
  - Keep existing: logo, menu, search, sticky, transparent, color scheme settings

### Step 5: Update `snippets/header-actions.liquid` styling
- **Keep all functionality** (cart drawer, account popover, cart-icon component)
- **Update CSS:** Style the icon buttons for the dark background (white color, proper sizing to match 24px icon spec)
- **Update cart bubble:** Change background color to use `settings.brand_congress_blue_default` (gold #e6b236) and white text

### Step 6: Update `sections/header-group.json`
- Remove the `header_announcements` section (replaced by the top bar built into header.liquid)
- Keep the `header_section` entry
- Update header settings to match new defaults (logo left, menu right, etc.)

### Step 7: Update `sections/header-announcements.liquid`
- Keep the file but it won't be used in the header group anymore (the top bar replaces it)
- No modifications needed — it can be re-added by merchants if they want an announcement bar in the future

## Schema Design (for header.liquid)

```
Settings:
  // Top bar
  - header: "Top bar"
  - checkbox: show_top_bar (default: true)
  - text: top_bar_link_1_text ("Arenus Ambassadors")
  - url: top_bar_link_1_url ("/")
  - text: top_bar_link_2_text ("Become a Dealer")
  - url: top_bar_link_2_url ("/")
  - checkbox: show_country_selector (default: true)
  - color: top_bar_bg_color (#191919)
  - color: top_bar_text_color (#dddddd)
  - color: top_bar_border_color (#45484a)

  // Pill buttons
  - header: "Quick links"
  - text: pill_1_text ("Colic Assurance Program")
  - url: pill_1_url ("/")
  - text: pill_2_text ("Our Blog")
  - url: pill_2_url ("/")

  // Main header
  - color: header_bg_color (#191919)
  - color: header_text_color (#ffffff)
  - color: cart_badge_color (#e6b236)
  - color: cart_badge_text_color (#ffffff)

  // Keep existing settings:
  - logo_position, menu settings, search, sticky, transparent, border, color schemes
```
